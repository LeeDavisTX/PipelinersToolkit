# Pipeliners Toolkit

React/Vite web application for the Pipeline Material Description Tool.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This project is configured with `base: '/PipelinersToolkit/'` in `vite.config.js` for GitHub Pages at:

https://leedavistx.github.io/PipelinersToolkit/
